# Видеохостинг - Инструкция по запуску

## Требования
- XAMPP (Apache + MySQL + PHP)
- Браузер с поддержкой HTML5

## Установка и запуск

### Шаг 1: Установка XAMPP
Если XAMPP еще не установлен:
1. Скачайте XAMPP с официального сайта: https://www.apachefriends.org/
2. Установите XAMPP в стандартную директорию (обычно `C:\xampp`)

### Шаг 2: Копирование проекта
1. Скопируйте все файлы проекта в директорию `C:\xampp\htdocs\video_hosting\`
   (или создайте папку с любым названием в `htdocs`)

### Шаг 3: Запуск серверов
1. Откройте XAMPP Control Panel
2. Запустите Apache (кнопка "Start")
3. Запустите MySQL (кнопка "Start")

### Шаг 4: Создание базы данных
1. Откройте браузер и перейдите по адресу: `http://localhost/phpmyadmin`
2. В левом меню нажмите "New" (Создать новую базу данных)
3. Или перейдите на вкладку "SQL"
4. Скопируйте содержимое файла `database.sql` и вставьте в SQL-окно
5. Нажмите "Go" (Выполнить)

### Шаг 5: Настройка подключения к БД
Если у вас нестандартные настройки MySQL:
1. Откройте файл `config.php`
2. При необходимости измените параметры:
   - `DB_HOST` - обычно `localhost`
   - `DB_NAME` - `video_hosting`
   - `DB_USER` - обычно `root`
   - `DB_PASS` - обычно пустая строка `''`

### Шаг 6: Запуск приложения
1. Откройте браузер
2. Перейдите по адресу: `http://localhost/video_hosting/` (или имя вашей папки)
3. Должна открыться главная страница видеохостинга

## Структура проекта

- `index.php` - Главная страница со списком видео
- `login.php` - Страница входа
- `register.php` - Страница регистрации
- `upload.php` - Страница загрузки видео
- `video.php` - Страница просмотра видео
- `admin.php` - Админ-панель
- `api.php` - API для обработки запросов (лайки, комментарии)
- `config.php` - Конфигурация и подключение к БД
- `functions.php` - Вспомогательные функции
- `style.css` - Стили оформления
- `script.js` - JavaScript для интерактивности
- `database.sql` - SQL скрипт для создания БД
- `uploads/videos/` - Папка для загруженных видео (создается автоматически)

## Функционал

✅ Регистрация и авторизация пользователей
✅ Разграничение по ролям (гость, пользователь, администратор)
✅ Загрузка видеороликов
✅ Лайки и дизлайки
✅ Комментарии к видео
✅ Просмотр списка видео для администратора
✅ Установка ограничений на видео (для администратора)

## Технические детали

- **Backend:** PHP (нативный, без фреймворков)
- **Frontend:** HTML, CSS, JavaScript (ванильный JS)
- **База данных:** MySQL
- **Хранение видео:** Локальная файловая система (папка `uploads/videos/`)

## Безопасность

- Пароли хешируются с помощью `password_hash()`
- Используется подготовленные SQL-запросы (защита от SQL-инъекций)
- Данные экранируются при выводе (защита от XSS)
- Проверка авторизации на защищенных страницах

## Возможные проблемы

**Ошибка подключения к БД:**
- Убедитесь, что MySQL запущен в XAMPP
- Проверьте настройки в `config.php`

**Видео не загружается:**
- Проверьте права на папку `uploads/videos/`
- Убедитесь, что папка существует

**Ошибка 404:**
- Проверьте правильность пути в адресной строке
- Убедитесь, что файлы находятся в `htdocs`
